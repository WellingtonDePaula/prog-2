# 📚 Programação II

Repositório com exercícios e projetos desenvolvidos durante a disciplina de **Programação II**.

## 🛠️ Tecnologias

- Python 3

## 📁 Estrutura

```
prog-2/
├── exercicios/   # Listas de exercícios
└── projetos/     # Projetos e trabalhos
```

## ▶️ Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/WellingtonDePaula/prog-2.git
   ```

2. Execute um arquivo:
   ```bash
   poetry run python nome_do_arquivo.py
   ```
   ou
   ```bash
   python nome_do_arquivo.py
   ```
